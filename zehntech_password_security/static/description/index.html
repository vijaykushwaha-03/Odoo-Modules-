Password Expiration

Administrators can set a password expiration policy that requires users to change their passwords after a specific time period, such as 60 days. This feature ensures that passwords are regularly updated to maintain security.

Password Complexity Requirements

Enforces the requirement for passwords to include various character types for added complexity. Administrators can specify the minimum number of lowercase, uppercase, numeric, and special characters.

User Notifications

If a user’s password is set to expire on October 10, they will receive an email notification on October 5 to October 10 reminding them to change their password.

Password Logs
